# defold-simple-keyboard
This is a pretty basic on-screen keyboard that includes:
- numbers: from 0 to 9
- letters: from a to z
- others: such as dot (.), space, delete and done (for finishing typing)

It is perfect for substituting an actual keyboard in mobile projects.
